3.更新（update）

==db.集合名称.update( query ,update,{multi: boolean})== 参数query:查询条件，类似sql语句update中where部分 参数update:更新操作符，类似sql语句update中set部分 参数multi:可选，默认是false，表示只更新找到的第一条记录，值为true表示把满足条件的文档全部更新

方式 对应语句 备注
全文档更新1 db.stu.update({name:'小明'},{name:'李明'}) 小明全部改为李明
全文档更新2 db.stu.update({age:10},{name:'李明'}) 改后年龄这个属性就没有了
指定属性更新 db.stu.update({age:10}，{$set :{name:'李明'}}) 改后年龄依然存在
修改多条匹配到的数据 db.stu.update({age:10},{$set:{name:'李明'}}，{multi:true})

5.删除

  • 语法：db.集合名称.remove(query,{justOne: boolean})

    参数query:可选，删除的文档的条件 参数justOne:可选，如果设为true或1，则只删除一条，默认false，表示删除多条

方式 对应语句

7.管道

文档处理完毕后，通过管道进行下一次处理常

管道 作用
$group 将集合中的文档分组，可用于统计结果
$match 过滤数据，只输出符合条件的文档
$project 修改输入文档的结构，如重命名、增加、删除字段、创建计算结果
$sort 将输入文档排序后输出
$limit 限制聚合管道返回的文档数
$skip 跳过指定数量的文档，并返回余下的文档
$unwind 将数组类型的字段进行拆分
表达式 作用
$sum 计算总和，$sum:1 表示以一倍计数
$avg 计算平均值
$min 获取最小值
$max 获取最大值
$push 在结果文档中插入值到一个数组中
$first 根据资源文档的排序获取第一个文档数据
$last 根据资源文档的排序获取最后一个文档数据

7.聚合

.
1.group 例子：分别求男生，女生的总人数
  db.stu.aggregate(
  {$group:{
  _id:'$gender',
  count:{$sum:1},
  avg:{$avg:'$age'}
 }}) 		1._id表_示分组的依据，使用某个字段的格式为'$字段'
2.count,avg 为自己命名的文档，先按gender进行分组
3.统计分组后每组的个数(按一倍统计) ，然后求每组年龄的平均值
2.group by null db.stu.aggregate(
  {$group:{
  _id : null,
  count : {$sum:1}
 }}) 		group by null,将集合中所有文档分为一组， 统计该组的个数
3.透视数据 $push
db.stu.aggregate(
  {$group:{
   _id : '$gender',
   name : {$push:'$name'}
  }}) 		按gender分组，并且打印每组数据中的name值
4. $project 查询男生、女生人数，输出人数
db.stu.aggregaate(
  {$group : {_id:'$gender',counter:{$sum:1}},
  {$project : {_id:0,counter:1}}
  }) 		按性别分组，得到id和count属性，通过$project控制分组后id不显示，counter显示
5.$sort 查询男生、女生人数，按人数降序
db.stu.aggregate(
  {$group : {_id:'$gender', counter:{$sum:1}},
  {$sort : {counter:-1}}
  }) 		此时$sort 的作用是，把分组后得到两个数（男生，女生人数）进行降序排列(-1降序，1升序)
6.$limit, $skip 统计男生、女生人数，按人数升序，取第2,3条数据
db.stu.aggregate(
  {$group : {_id:'$hometownr', counter:{$sum:1}},
  {$sort : {counter:1}},
  {$skip : 1},
  {$milit : 2}
  })
 行号： 2.统计男生，女生人数
 3.按人数升序排列
 4.跳过第一条数据
 5.在4的基础上在往下取两条数据
7.$unwind 构造数据

db.ts.aggregate({$unwind : '$size'})
将插入的数据按size拆分，拆分的结构为3条数据，_id,item,的值不变，size分别为S,M,L
