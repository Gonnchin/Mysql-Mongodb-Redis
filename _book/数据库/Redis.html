3种常用数据库

Redis数据库

  1. redis中数据库没有名字，默认16个，用0-15表示
  2. redis是key-value 的数据结构，每条数据都是一个键值对
  3. 键：字符串类型，键不能重复
  4. 值：字符串，哈希(hash),列表，集合(set)，有序集合(zset) （当关闭数据库后，下次查询对于很多数据肯定不能完全记住其 类型 这时就需要通过type 键名，查询键的类型然后用相关命令在去执行）

1.string,键值的设置，获取，键命令（键的查找）

操作 对应语句 备注
设置/修改键值 set name Tom 设置键为name 值为Tom ,若键name存在则修改其值
设置键值及过期时间(秒) setex name 6 Tom 设置过期时间为3秒，3秒后name对应的值将为空
设置多个键值 mset name Tom hometown 浙江 同时设置name和hometown键，对应值分别为Tom,浙江
追加键值 append hometown 杭州 在上一步hometown值的基础上追加，现在hometown对应键值为浙江杭州
获取键值 get name 获取键name对应的键值
由多个键获取多个键值 mget name age 将会获取到键name,age 对应的键值
删除键值 del name age 键name,age及其对应的键值都会被删除
操作 对应语句 说明
查找键1 keys * 查看所有键
查找键2 keys n 查找键名中包含字母n的键
判断键是否存在 exists name 判读键name是否存在
查看键值的类型 type name 查看键name对应键值的类型
设置的过期时间(s) expire name 10 10s后键name将会被删除，无论设置前是否有键值
查看有效时间(s) ttl name 查看键name的有效时间

2.hash

hash用于存储对象，对象的结构为属性、值，值的类型为string

对应语句
增加/修改单个属性 hset py1 name Tom 设置键py1的属性name值为Tom
设置多个属性 hmset py1 name Tom age 12 设置py1属性name,age的值分别为Tom,12
获取所有属性的值 hvals py1
获取键所有属性 hkeys py1 获取键py1的所有属性
获取键的所有属性及值 hgetall py1 获取键py1的所有属性及值
获取一个属性的值 hget py1 name 获取键py1属性name的值
获取多个属性的值 hmget py1 name age 获取键py1属性name和age的值
删除键的属性 hdel py1 name age 删除j键py1的name和age属性

3.list列表

  • 列表的元素类型为string,按照插入顺序排序
对应语句
左侧插入数据 lpush py2 a b 从键为py2的列表左侧加入数据a,b
在右侧插入数据 rpush py2 c d 从键为py2的列表右侧加入数据c,d
在指定元素的前或后插入新元素 linsert py2 before c f 在键为py3的列表中元素c前加入f
获取(索引从左开始) lrange py2 0 -1 获取键为py2的列表所有元素 ,0表示第一个元素，-1表示最后一个元素
修改 lset py2 1 g 将键py2的列表中下表为1(索引从左开始) 的元素改为g
删除指定元素 rem py2 -2 a 从py2列表右侧开始删除a ,一共删除2个a

  • rem key count value ：count表示删除的个数，大于0从头往尾移除，小于0从尾往头移除，等于0全部删除

  • 备注：返回列表里指定范围内的元素,start、stop为元素的下标索引,索引从左侧开始，第一个元素为0,索引可以是负数，表示从尾部开始计数，如-1表示最后一个元素

4.set集合

  • 无序集合,元素为string类型,元素具有唯一性，不重复,对于集合没有修改操作 （可用于去重，例如从网站上下载数据出现故障，需从头开始下载，若此时是通过集合进行储存则不会出现重复下载—）
操作 语法 例子 说明
添加元素 sadd key menber1 menber2... set py3 a b c 向键py3的集合中添加a,b,c
获取所有元素 smembers key smember py3 获取py3的集合中所有的元素
删除指定元素 srem key member srem py3 a 删除py3的集合中a 这个元素

5.zset有序集合

  • 和set相比，是有序集合
  • 每个元素都会关联一个double类型的score，表示权重，通过权重将元素从小到大排序
操作 例子 说明
添加 zadd py4 3 a 4 c 向集合py4中添加a,b两个元素，设置其权重分别为3,4 ，存储的顺序为c,a
获取 zrange py4 0 2 在集合py4中从下标为0的数据开始获取到下标2结束
返回score在一定范围内的元素 zrangebyscore py4 2 6 获取py4中权限值在2和6之间的元素
返回成员member的score元素 zscore py4 a 获取集合py4中a的权重值
删除指定元素 zrem py4 a 删除集合py4中的元素a
删除权重在指定范围的元素 zremrangebyscore py4 2 6 获取集合py4中权限值在2和6之间的元素

zadd key score1 member1 score2 member2 ...
