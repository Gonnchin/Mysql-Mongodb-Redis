例子

模糊查询 1.查询姓李的女学生: select * from students where gender='女' and name like '李% ';

空判断 3.查询年龄为空的学生：select * from students where age is null;

3.2 排序order by 查询

语法 select * from 表名 order by 列1 asc|desc [,列2 asc|desc ,...]
将查询出的数据先按列1进行排序，如果列1中某些值相同在按列2排序...
asc升序，desc降序，默认升序
例子 查询男生信息，按学号升序，学号相同时按年龄升序排
select * from students where gender='男' order by id asc,age desc;

3.3 聚合函数查询

总数count(*) 计算的是总行数
查询男生的个数
select count(*) from students where gender='男';
最大/小值max(列)/min(列) 查询女生的编号最大值
select max(id) from students where gender='女';
求和sum(列) 查询所有学生年龄的平均值
select sum(age)/count(*) from students;
平均值avg(列) 查询所有学生年龄的平均值
select avg(age) from students;

+--------+----------+

5.SQL语句的强化

  1. 求所有电脑产品的平均价格,并且保留两位小数

    select round(avg(price),2) as avg_price from goods;

  2. 查询所有价格大于平均价格的商品，并且按价格降序排序

    select id,name,price from goods 
where price > (select round(avg(price),2) as avg_price from goods) 
order by price desc;

  3. 查询类型cate_name为 '超极本' 的商品名称、价格

    select name,price from goods where cate_name = '超级本';

  4. 显示商品的种类

    select cate_name from goods group by cate_name;

  5. 显示每种商品的平均价格

    select cate_name,avg(price) from goods group by cate_name;

  6. 查询每种类型的商品中 最贵、最便宜、平均价、数量

    select cate_name,max(price),min(price),avg(price),count(*) from goods group by cate_name;

  7. 查询每种类型中最贵的电脑信息

    select * from goods
inner join 
 (
     select
     cate_name, 
     max(price) as max_price, 
     min(price) as min_price, 
     avg(price) as avg_price, 
     count(*) from goods group by cate_name
 ) as goods_new_info 
on goods.cate_name=goods_new_info.cate_name and goods.price=goods_new_info.
